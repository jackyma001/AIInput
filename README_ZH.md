# AIInput - 智能语音输入助手

AIInput 是一个高效的 Windows 语音录入工具。它可以监听全局热键，将你的语音实时转录为文字，并自动输入到当前光标所在的位置。

## 核心特性 ✨

- **推键即说 (Push-to-Talk)**：按住 `Ctrl + Win` 开始录音，松手即刻完成转录并输入，极速体验。
- **极速识别**：搭载 `faster-whisper` AI 引擎，比原版 Whisper 快 4-5 倍。
- **视觉反馈**：屏幕底部弹出精致的半透明波形动画窗口，实时显示音量波动。
- **智能纠错**：强制输出**简体中文**，避免繁简混杂，提高办公效率。
- **开机自启**：右键点击托盘图标即可轻松开启开机自动启动。
- **架构稳定**：全软件统一使用 PyQt5 引擎，告别闪退，运行稳如泰山。

## 安装步骤 🛠️

1. **环境要求**：Python 3.8+ (推荐 3.12)。
2. **下载代码**：
   ```bash
   git clone https://github.com/jackyma001/AIInput.git
   cd AIInput
   ```
3. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
   *注意：如果遇到 VAD 报错，请确保已安装 `onnxruntime`。*

## 使用说明 💡

1. **启动程序**：
   ```bash
   python src/main.py
   ```
2. **开始输入**：
   - 将光标点击在任何你想输入文字的地方（如 Word、微信、代码编辑器）。
   - **按住** `Ctrl + Win` 键并开始说话。
   - **松开** 按键，文字将瞬间出现在光标处。

## 配置与优化 ⚙️

你可以通过修改 `src/config.py` 来进一步自定义：
- `MODEL_SIZE`: 默认为 `tiny`（最快）。追求更高准确度可改为 `base` 或 `small`。
- `DEVICE`: 如果你有 NVIDIA 显卡，改为 `cuda` 可获得飞一般的速度。
- `LANGUAGE`: 默认为 `zh`（简体中文引导）。

## 故障排查 🔍

- **没有声音？** 请检查 Windows 默认麦克风设置。
- **程序关闭？** 查看根目录下的 `app.log` 文件，里面记录了详细的运行日志和错误信息。
